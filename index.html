Men ismim Bahriddinov Samandar 2003-yil 31-iyul kuni Farg'ona viloyati Bag'dod tumanida ziyoli oilada dunyoga keldim. 2010-yil Bag'dod tumanidagi 1-sonli umumiy o'rta ta'lim maktabiga 1-sinf bo'lib qabul bo'ldim. 2021-yil 11-sinf bo'lib a'lo baholarga maktabni tugatdim. Keyinchali IT ga qiziqanim sababli IT yo'nalishidan biri Frontend kursida o'qidim va yaxshi tamomladim.

S e r v i c e

Desktop xizmati - bu degani ya'ni odatda birinchi bo'lib websaytlar desktop ya'ni kompyuter o'lchamiga tuzuladi. Keyinchali websaytlar boshqa o'lchamlarga moslanadi.

Coder xizmati - bu xizmat websayt uchun kerakli dasturlar orqali kod yoziladi. Agarda kod yozilmasa hech qanday websayt hosil bo'lmaydi bo'sh varoq ka'bi bo'sh turaveradi.

Moslashuv xizmati - bu xizmat desktop, mobile versiya uchun websayt yoziladi ya'ni moslashtiradi har xil qurulma uchun va bu xizmat kundan kunga rivojlanyapti.

Statiska - birinchi o'rinda websaytingiz qaysi texnologiyada yozilishiga e'tibor bering chunki hozirda dasturlash texnologiyalari kundan kunga raqobat ka'bi o'sib borilyapti.

M y     s k i l l s

HTML

HTML - (Hypertext Markup Language) - bu dasturlash tili emas. Websaytlar birinchi bo'lib html orqali strukturasini tuzub olinadi. Men bu texnologiyani o'qish davomida yaxshi o'zlashtirganman deb hisoblayman.

CSS

CSS - (Cascading Style Sheets) - bu texnologiyada HTML orqali tuzulgan strukturaga style beriladi ya'ni bezaklar turli xil hajm va ranglar orqali websaytingizni ya'nada chiroyli qilishingiz mumkun.

SASS

SASS - (Syntactically Awesome Stylesheets) - bu Css kabi bir xil vazifani bajaradi lekin qulaylik bo'yicha Sass yaxshiroq junki yozga style kodingizni tartibli va tushunarli qilib saqlaydi va o'rganish juda oson.

Bootstrap

Bootstrap - bu mobile qurilmalar uchun moslashuvchan hamda frontend dasturlash uchun mo'ljalangan bepul va ochiq kodli framework. Bu framework tipografiya, formalar, tugmalar, navigatsiya va boshqa interfeyslarni o'z ichiga oladi. Bu frameworkda JavaScriptni ham foydalansa bo'ladi.

JavaScript

JavaScript - JS nomi bilan ham qisqartiladigan ECMAScript spesifikatsiyaga mos keladigon yuqori darajali hamda aniq talqin qilingan dasturlash tili. Bu dasturlash tili orqali websaytingizni turli xil joylarini harakatga, animation yasashingiz mumkun.

ReactJS

ReactJS - React-dan bitta sahifa va mobil ilovalarni ishlab chiqish uchun foydalanish mumkin. Uning maqsadi yuqori tezlik, soddalik va miqyoslilikni ta'minlashdir . Foydalanuvchi interfeyslarini ishlab chiqish uchun kutubxona sifatida React ko'pincha MobX, Redux va GraphQL kabi boshqa kutubxonalar bilan birga ishlatiladi.

Figma

Figma - bu turli xil maqsadlarda grafikani ishlab chiqish vositasi. Afishalar tashrif qog'ozlar, websaytlar, Android ilovalari, IOS ilovalari yoki ish stoli dasturlari. O'zining asosiy va bepul versiyasida u allaqachon professional va jozibali interfeyslarni yaratish uchun zarur va zarur narsalarni taqdim etadi.

Redux

Redux - ilova holatini boshqarish uchun ochiq manbali JavaScript kutubxonasidir. Ko'pincha frontend ishlab chiqish uchun React yoki Angular bilan birgalikda ishlatiladi. Kontekst orqali saqlash ma'lumotlarini uzatishni sezilarli darajada soddalashtirish uchun bir qator vositalarni o'z ichiga oladi
